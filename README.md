# Expense Tracker (Static Frontend)

This is a static front-end version of an Expense Tracker project.  
It demonstrates how expenses can be displayed, categorized, and visualized with charts using only HTML, CSS, and JavaScript.

## Features
- Responsive layout with cards for expenses
- Chart visualization using Chart.js
- Light and dark theme toggle
- Simple navigation between Home and Add Expense pages

## Project Structure
index.html        - Main dashboard with expenses and chart  
add_expense.html  - Form page to add a new expense (demo only)  
styles.css        - Styling and theme variables  
script.js         - Chart.js setup and theme toggle  
README.md         - Project documentation  

## Deployment
This project is deployed using GitHub Pages.  
Live demo: https://nadimintymanaswini3003.github.io/expense-tracker

## Note
This is a static demo. Data is hardcoded for showcase purposes.  
The backend version with Flask and database is not included here, since GitHub Pages only supports static files.
