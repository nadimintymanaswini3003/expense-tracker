#  Expense Tracker

A simple and engaging **Expense Tracker web app** built with **Flask + SQLite**.  
It helps you record expenses, view totals, and visualize spending with a **pie chart dashboard**.  
Includes a **dark/light theme toggle** and a clean teal‑based UI.

---

##  Features
- Add expenses with amount, category, and description
-  View all expenses in card layout
-  Pie chart breakdown by category (Chart.js)
-   Dark/Light theme toggle
- Modern teal palette with hover animations

---

## Tech Stack
- **Backend:** Flask (Python), SQLite
- **Frontend:** HTML, CSS, Chart.js
- **Database:** SQLite (auto‑created `expenses.db`)
